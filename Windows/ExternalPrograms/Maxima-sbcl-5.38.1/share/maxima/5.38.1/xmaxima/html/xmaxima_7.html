7. Getting Help

Xmaxima can be used to access Maxima's documentation in several different forms. The describe of Maxima should work exactly as it does in Maxima. The Help menu has a link to the local copy of the manual, which is distributed with Maxima. In that menu there is also a link to the Website of Maxima, where some other documentation can be found.
