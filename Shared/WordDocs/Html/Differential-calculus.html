Differential calculus

In differential calculus, it is generally a good idea to start by defining the function(s) to be worked with

\[Define\! :f\left(x\right)=x^{2}-3x+1\]

Now you can calculate the derivative, for example:

\[f'\! \left(x\right)=2\cdot x-3\]

Or find the differential quotient in a single point:

\[f'\! \left(2\right)=1\]

The ' prime tags can be typed in several different ways on a keyboard. The different primes are all similar:    ’ ´ `

WordMath supports all ways to enter prime tags, but it is recommended to use the one that is just to the left of the Enter button on the same button as *. It looks best and is the easiest to access.

You don't need to define a function to differentiate it. The following notation forms are also supported:

\[\left(x^{2}+2x-1\right)'\! \left(x\right)=2\cdot x+2\]

\[\left(x^{2}+2x-1\right)' =2\cdot x+2\]

The expression does not have to be a function, but then the variable must be x.

\[Define\! : y=x^{2}\]

\[y' =2\cdot x\]

In relation to the trigonometric functions, it is important that WordMath is set to calculate in radians

\[\left(\sin \left(x\right) \right)' =\cos \left(x\right)\]

If WordMath is not set to radians, it corresponds to the function  \(\sin \left(\frac{\pi }{180}\cdot x\right)\)  being differentiated, and then the result is:

\[\left(\sin \left(x\right) \right)' =\frac{\pi\cdot\cos \left(x\right)}{180}\]

WordMath also supports 2nd and 3rd derivatives etc.

\[f''  \left(x\right)=2\]

 

Differential notation is also supported, but there is a problem. If you just type dy/dx it doesn't work:

\[\frac{dy}{dx}=\frac{dy}{dx}\]

This is because WordMath does not reserve the variables dy and dx. WordMath needs to know that you have not just entered an expression, where there are two variables called dy and dx.

This can be done in two different ways.

1.      Differential d
Word supports entering a special d, which can be used for dy/dx notation. This d is entered by typing '\dd' followed by a space. And then it works:

\[\frac{dy}{dx}=2\cdot x\]

It looks exactly like a normal d, but there is a difference.

 

2.      The setting: "Use regular d as differential d"

It is also possible to get it to work by typing a regular d. This is done by opening Settings > Notation and ticking Use regular d as differential d

\[\frac{dy}{dx}=2\cdot x\]

This setting can possibly cause problems in some contexts where d is part of a common equation. However, this is very unlikely.

 

Other examples of supported notation:

 \(\frac{df\left(x\right)}{dx}\)   ,   \(\frac{d}{dx} f\left(x\right)\)    ,    \(\frac{d}{dx} \left(x^{2}+2x\right)\)    ,    \(\frac{d^{2}}{dx^{2} } \left(x^{2}+2x\right)\)

 

 

Partial derivatives

The differential symbol used for partial derivatives: ∂ - is entered with the shortcut "\partial"

Here are some examples of notation that can be used with the partial derivative symbol without defining a function:

\[\frac{\partial }{\partial x} \left(x^{2}\cdot y\right)=2\cdot y\cdot x\]

\[\frac{\partial ^{2}}{\partial x\partial y} \left(x^{2}\cdot y\right)=2\cdot x\]

If you have defined a function of two variables, you can also use the subscript notation to determine partial derivatives:

\[Define\! : f\left(x,y\right)=x^{2}+3x\cdot y-2y+1\]

\[f_{y'}\! \left(x,y\right)=3\cdot x-2\]

\[f_{yx}''  \left(x,y\right)=3\]

\[\left(3x^{2}+2y\cdot x\right)_{x}' =6\cdot x+2\cdot y\]

\[\frac{\partial ^{2}}{\partial x\partial y} f\left(x,y\right)=3\]

\[\frac{\partial ^{2} f\left(x,y\right)}{\partial x\partial y}=3\]

 

Example exercise: Tangent determination

Determine the tangent of the function  \(f\left(x\right)=\frac{1}{2} x^{2}-2x+1\) in the point  \(\left(4, f\left(4\right)\right)\)

First define the function and the x-value of the point

\[Define\! : f\left(x\right)=\frac{1}{2} x^{2}-2x+1, x_{0}=4\]

Next, the formulae for the tangent's equation is retrieved by searching for 'tangent' in the formula booklet. (Alt+F)

\[y=f'\! \left(x_{0} \right)\cdot \left(x-x_{0} \right)+f\left(x_{0} \right)=2\cdot x-7\]

The tangent is found by performing Calculate on the formula. Note that it has not been necessary to determine \(f’\left(x\right)\) , just define the components that are in the formula.

 

Nabla operator and functions of two variables

There is support for the nabla operator.
Enter it as follows:

\nabla  →    \(\nabla \)

Examples:

\[Define\! : f\left(x,y\right)=x^{2}\cdot y\]

Calculation of the gradient as a vector function and in a specific point:

\[\nabla f\left(x,y\right)=\left(\begin{matrix}2\cdot y\cdot x\\x^{2}\end{matrix}\right)\]

\[\nabla f\left(1, 2\right)=\left(\begin{matrix}4\\1\end{matrix}\right)\]

Example of the Laplace operator:

\[\nabla ^{2} f\left(x,y\right)=\left(\begin{matrix}2\cdot y\\0\end{matrix}\right)\]
