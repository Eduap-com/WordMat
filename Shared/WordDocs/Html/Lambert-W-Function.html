Lambert W Function

The Lambert W function is the inverse function of

\[f\left(x\right)=x\cdot e^{x}\]

The function is not injective and therefore comes in two 'branches':  \(W_{0} \left(x\right)\) and  \(W_{-1} \left(x\right)\)

These two functions are therefore reserved in WordMath.

They will only appear if WordMath is set to specifically calculate exactly, otherwise they will be rounded numerically.

The Lambert W function is used by WordMath to solve equations such as

Example 1:

\[x\cdot e^{x}=4\]

                              \(\Updownarrow \)              The equation is solved for x using WordMath.

\[x=W_{0} \left(4\right)\approx 1,\!202168\]

Example 2:

\[3^{x}=5x+1\]

                              \(\Updownarrow \)              The equation is solved for x using WordMath.

\[x=2,\!298454   \quad\vee\quad     x=0\]
