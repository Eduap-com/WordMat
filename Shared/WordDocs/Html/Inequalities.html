Inequalities

You solve inequalities in the same way as equations.

Example:

\[x^{2}\geq 9\]

                              \(\Updownarrow \)                                           The inequality is solved for  x by WordMat.

\[x\leq -3  \quad\vee\quad    x\geq 3\]

More complex example:

                              \(\Updownarrow \)                                           The inequality is solved for  x by WordMat.

\[\left(x\geq -1   \quad\wedge\quad     x\leq 0\right)  \quad\vee\quad    x\geq 2\]

Note:

1.       Simultaneous equation with inequalities is not supported.

2.       WordMath is not always as good at solving inequalities as equations. If it is not possible to solve an inequality, the following can be tried:

      Change the CAS engine to GeoGebra, which is often better for inequalities.

      Solve the inequality as an equation and show a graph of both the right and left sides in a coordinate system, so you can see which side is highest between the solutions.

      GeoGebra and Graph can display the solution graphically. Select the inequality and press Show Graph (Alt + P)– then the solution will be shown graphically.
Example: graphical solution of  \(x^{2}\geq 9\)       solutions are in the purple range.
