Vectors

To work with vectors in WordMath, you need to be able to enter a vector in a math field. There are several ways to do this, but it's recommended to use the keyboard rather than the templates in the equation menu – it's both faster and easier once you've tried it a few times.

To enter a vector
Enter the starting bracket, ending bracket, and then press space to get the following

\[( \Box )\]

Then press the left arrow key and press Enter – then you get a vector with two components:

\[\begin{pmatrix} \Box \\ \Box \end{pmatrix}\]

If you press Enter one more time, you get a 3D vector:

\[\begin{pmatrix} \Box \\ \Box \\ \Box \end{pmatrix}\]

Examples of vector calculations

·         Vector addition and multiplication with a constant

\[\left(\begin{matrix}1\\3\end{matrix}\right)+2\cdot \left(\begin{matrix}-1\\2\end{matrix}\right)=\left(\begin{matrix}-1\\7\end{matrix}\right)\]

·         The length of a vector

\[\left |\left(\begin{matrix}2\\3\end{matrix}\right)\right |=\sqrt{13}\]

The vertical lines are entered on Windows by holding Alt Gr + the button just to the left of the backspace.  On Mac: Option + i.

·         Scalar product

\[\left(\begin{matrix}1\\4\end{matrix}\right)\cdot \left(\begin{matrix}-3\\2\end{matrix}\right)=5\]

·         Ortogonal vector

\[\widehat{\left(\begin{matrix}3\\5\end{matrix}\right) }=\left(\begin{matrix}-5\\3\end{matrix}\right)\]

The cross-vector hat is entered with \hat

·         Determinant

\[det\left(\left(\begin{matrix}1\\3\end{matrix}\right),\left(\begin{matrix}2\\3\end{matrix}\right)\right)=-3\]

·         Cross-product (only for vectors in space(3D))

\[\left(\begin{matrix}2\\3\\4\end{matrix}\right)\times \left(\begin{matrix}-1\\6\\-2\end{matrix}\right)=\left(\begin{matrix}-30\\0\\15\end{matrix}\right)\]

The cross is entered with \times

Using vector arrows
It is recommended to define vectors with an arrow above the variable.

·         To type a with arrow:
Type a\vec and press space twice:   \(\vec{a}\)

·         Alternatively, you can use \rhvec for a different vector arrow:  \(\overset{\rightharpoonup}{a}\)

If vectors are not defined with an arrow, you should be aware that WordMath may otherwise interpret the expression incorrectly – e.g. in the case of dot product.

Examples with defined vectors

\[Define\! :\vec{a}=\left(\begin{matrix}2\\3\end{matrix}\right)  ,\vec{b}=\left(\begin{matrix}-1\\2\end{matrix}\right)\]

·         Orthogonal vector

\[\widehat{\vec{a}}=\left(\begin{matrix}-3\\2\end{matrix}\right)\]

Here the orthogonal vector is found using the hat operator. Typed a\vec\hat

 

·         Angle between vectors

Use the formula collection and search for "vector". What is inserted in this case is a bit atypical, as the formula is inserted once and just before the formula the two vectors are defined. The idea is that you change the coordinates of the two vectors to the specific problem.

\[Define\! :\vec{a}=\left(\begin{matrix}2\\3\end{matrix}\right),\vec{b}=\left(\begin{matrix}-1\\2\end{matrix}\right)\]

\[\cos \left(v\right)=\frac{\vec{a}\cdot\vec{b}}{\left |\vec{a}\right |\cdot \left |\vec{b}\right | }\]

                              \(\Updownarrow \)              The equation is solved for v by WordMat.

\[v=60,\!25512\]

                            

·         Projection of a vector

\[\vec{b_{a} }=\frac{\vec{a}\cdot\vec{b}}{\left |\vec{a}\right |^{2}} \vec{a}\approx \left(\begin{matrix}0,\!6153846\\0,\!9230769\end{matrix}\right)\]

\[\left |\vec{b_{a} }\right |=\frac{\left |\vec{a}\cdot\vec{b}\right |}{\left |\vec{a}\right |}\approx 1,\!1094\]

Vector
Vector equations are actually systems of equation that WordMath can solve automatically.

·         Example

\[\left(\begin{matrix}x+y\\x-y\end{matrix}\right)=\left(\begin{matrix}1\\2\end{matrix}\right)\]

                              \(\Updownarrow \)              The equation is solved for y by WordMath.

\[y=-\frac{1}{2}   \quad\wedge\quad     x=\frac{3}{2}\]

It can be a little sensitive to which variable is chosen. In this case, you have to choose y for it to work. If this does not succeed, it may be necessary to convert the vector equation to a system of equations, and then solve the system by selecting the equations.

 

·         Intersection between line and plane.

Select the two math fields and press Solve.

\[\left(\begin{matrix}x\\y\\z\end{matrix}\right)=\left(\begin{matrix}1\\-1\\6\end{matrix}\right)+t\cdot \left(\begin{matrix}2\\-2\\4\end{matrix}\right)\]

\[4x-6y+10z-26=0\]

                              \(\Updownarrow \)              The system of equations is solved for x,y using WordMath's 'Solve Equations' function,

\[t=-\frac{11}{15}   \quad\wedge\quad     z=\frac{46}{15}   \quad\wedge\quad     y=\frac{7}{15}   \quad\wedge\quad     x=-\frac{7}{15}\]

Square brackets
WordMath also supports vectors with square brackets. If the input uses square brackets, the output will also do so:

\[2\cdot \left[\begin{matrix}2\\3\end{matrix}\right]=\left[\begin{matrix}4\\6\end{matrix}\right]\]

 

 

Referencing elements in vectors/matrices

You can refer to elements of a vector or matrix in several ways:

·          \(\vec{v}[r]\left[k\right]\)

·          \(\vec{v}[r,k]\)

·          \(\vec{v}_{r ,k}\) (requires that "Subscript is list/matrix index" is turned on under "Settings > Notation". See Index / Subscript p. 13 )

where r is row number and k is column number (for vectors, k is always 1)

Example:

\[Define\! :\vec{v}=\left(\begin{matrix}1\\2\\3\end{matrix}\right)\]

\[\vec{v}[2]\left[1\right]=2\]

\[\vec{v}[2 , 1]=2\]

\[\vec{v}_{2 ,1}=2\]
