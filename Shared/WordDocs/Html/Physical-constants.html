Physical constants

Under the definitions menu, there is an item called Define physical constants. You can use this to insert definitions of the most common physical constants and most commonly used values of different physical properties . Select the constants to be used and press OK, then these will be inserted as definitions in the document.

Example:

\[Define\! :  c=299792458 m\cdot s^{-1};c_{vand}=4181 J\cdot kg^{-1}\cdot K^{-1};m_{sol}=1,\!98892\cdot 10^{30} kg\]

If you have WordMath+, you can find more physical constants and values in the physics formulae.
