Piecewise defined functions

WordMath can both make calculations and show graphs for piecemwise defined functions. Here's an example

\[Define\! : f\left(x\right)=\left\{\begin{matrix}-x   ,x< 0\\x^{2}       ,x\geq 0 \end{matrix}\right.\]

It is important that it is entered correctly. The individual curly brackets are inserted via the "equation menu / Brackets / Examples and stacks". It is also important that a comma is used to separate the function and the scope of definition. Use spaces around the comma to avoid it being interpreted as part of a number.

With the above definition in the document, the following can be calculated

\[f\left(-1\right)=1\]

\[f\left(2\right)=4\]

You can use different spellings to specify definition quantities. You can use  \(\quad\vee\quad \)  and \(\quad\wedge\quad \) , but you can also use Boolean operators: OR, AND, or, and.

Here is an example, where it is applied in the 2nd definition

\[Define\! : f\left(x\right)=\left\{\begin{matrix}2x+2  , 0< x< 3\\-x+3  , x\geq 3\quad\wedge\quad  x< 5\\x^{2}  ,x\geq 5\end{matrix}\right.\]

 

Piecewise functions can also be entered without definition, if it just needs to be plotted with GnuPlot or GeoGebra.

 

Typical mistakes are

·         Forgetting the comma

·         Forget space after the comma

·         Use two curly brackets instead of one
