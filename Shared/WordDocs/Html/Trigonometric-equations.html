Trigonometric equations

Let's look at the equation

\[\sin \left(x\right)=\frac{1}{2}\]

                              \(\Updownarrow \)              The equation is solved for x using WordMath.

\[x=30\]

WordMath shows only one solution to the equation by default, but in fact there are an infinite number of solutions. In most cases, however, it is only the first solution larger than 0 you are interested in.

In WordMath's settings, you can choose whether you want only one solution or all solutions for trigonometric equations. By default, it is set to one solution, as this is where you typically start in mathematics teaching. You will find the option on the CAS tab.

Let's see what happens when we change the setting:

\[\sin \left(x\right)=\frac{1}{2}\]

                              \(\Updownarrow \)              The equation is solved for x by WordMath.

\[x=360\cdot Z+30   \quad\vee\quad     x=360\cdot Z+150\]

 \(Z\) here represents an integer (..., -2, -1, 0, 1, 2, ...)

Typically, however, you do not need an infinite number of solutions, but solutions within a certain area. Here you can use assumptions that are explained in more detail in section 7 Definitions / Assumptions p. 40.

\[Define\! : 0\leq x\leq 180\]

\[\sin \left(x\right)=\frac{1}{2}\]

                              \(\Updownarrow \)              The equation is solved for x by WordMat.

\[x=30   \quad\vee\quad     x=150\]

 

Again: it is important to remember to delete assumptions.
