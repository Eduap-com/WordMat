Definition fields

A definition field is a math field that starts with  \(Define\! :\)
You can insert a definition field with the keyboard shortcut Alt + D or through the menu under "Def."

\[Define\! :\]

You enter your definition after the colon.
 Example:

\[Define\! : a=2\]

Now a has the value 2, so if a is included in a calculation, it just corresponds to 2.

\[3\cdot a=6\]

In the same way, you can define a function:

\[Define\! : f\left(x\right)=2\cdot x+1\]

\[f\left(2\right)=5\]

You can enter more than one definition in a definition field by separating with semicolons or commas.

\[Define\! : a=2, b=3\]

\[a\cdot b=6\]

You can also type "Define:" yourself, but it is recommended to use the keyboard shortcut, as it fails in case of spelling mistakes or, for example, spaces before colons.

There is also another type of definition field that uses a special definition equal sign instead of typing "Define:".

Example:

\[a:=2\]

The following 3 definition equal signs are supported:

≔    is just entered with a colon and an equal sign

 ≡    entered with \equiv

 \(\overset{{\tiny def}}{=}\)     entered with \defeq

As with define: you can also enter multiple definitions in a definition field when you apply definition equal signs:

\[a:=3 , b\equiv 4\]

\[a\cdot b=12\]

Notice how a and b have assumed new values, even though they are also defined above. If the same variable is defined twice, the most recent one will apply.

Deleting definitions

You can delete definitions so that they are no longer valid in the rest of the document. This is done by inserting a 'Delete definitions' field

 

This can be done via the menu, but easiest with the default keyboard shortcut Alt + S

 

 

It deletes all definitions. Remember to use it when you no longer need the definitions, as you can easily end up having to use the same variable further down in the document where the variable should not be defined. Also note that the speed of calculations can be affected if you use definitions a lot. If you remember to use 'delete definitions:' you avoid significantly affecting the speed in large documents.

You can also delete individual variables and functions by writing which variables and functions should be deleted after the colon. Example:

\[Delete definitions:a, f\]

In this example, 2 of the 3 definitions are deleted and we can see in the calculation that only one definition is still valid:

\[Define\! : a=2,b=3,f\left(x\right)=x^{2}\]

\[Delete definitions:a, f\]

\[a+b+f\left(x\right)=f\left(x\right)+a+3\]

In the WordMath menu, there is a Def button. If you press the top part of the button, it will display all the definitions that apply to the place where the cursor is located.

 

Variable names
Variables and functions in WordMath can be more than one letter and can contain numbers, but a name must not start with a number. A variable can also have subscript letters as part of the name.

Example:

\[car_{1}:=200000, car_{2}:=300000\]

\[\frac{car_{1}+car_{2}}{2}=250000\]

Variable names with Greek characters are also supported.

\[\beta :=2,\lambda :=3\]

Example of entering Greek letters:

\beta ->        \(\beta \) \lambda ->   \(\lambda \)
\Lambda ->    \(\Lambda \)

 

Important about definition fields

1.       You can't perform calculations in a definition field. The definitions are there so that they can be used in the calculations below the definitions.

2.       If there is an error in a definition, WordMath cannot perform calculations below that definition, as all calculations will fail. If this happens, WordMath will come up with an error message and mark the definition where the error is.

3.       It is allowed to place all the spaces you think are appropriate in the definition fields to make it more readable.

4.       When you make a calculation, WordMath searches up the document to find definitions written in definition fields. Only definitions that are listed before the calculation are used. If you define a variable or function multiple times, the most recent definition will apply.

5.       Note that a distinction is made between upper and lower case letters in variable and function names. I.e. f(x) and F(x) are not the same function.

One can also define vectors. See more about this in section 13 Vectors on p.86.

 

Definition of functions

As with variables, functions can be defined in different ways:

\[Define\! :f\left(x\right)=x^{2}+1\]

\[g\left(x\right):=2x+3 ;h\left(t\right)\equiv t+1\]

The definitions can then be used to calculate, for example:

\[g\left(2\right)=7\]

\[g\left(3\right)+\frac{h\left(6\right)}{f\left(2\right)} =\frac{52}{5}=10,\!4\]

Definitions can also be included in equations

\[f\left(x\right)=h\left(x\right)+g\left(1\right)\]

                              \(\Updownarrow \)              The equation is solved for x by WordMath.

\[x=-\frac{\sqrt{21}-1}{2}   \quad\vee\quad     x=\frac{\sqrt{21}+1}{2}\]

 

 

It is possible to define functions of several variables. You must then be aware of which list separator you have chosen.

\[Define\! : f\left(x;y\right)=x^{2}+y^{2}\]

\[f\left(2;3\right)=13\]

If semicolon (default) is set as listseparator you can also use comma, but WordMath determines the list separator based on context.
You can write f(x,y) but not f(2,3) unless you make space at either side of the comma: f(2, 3).

\[f\left(2, 3\right)=13\]

\[f\left(2,\!3\right)=syntax error\]

Here 2,3 is interpreted as the decimal number 2,3 and not as two numbers. Calculations fail as the function requires two numbers.

See also section 11 Infinitesimal calculus/Differential calculus on p. 74 for more on functions of two variables.
